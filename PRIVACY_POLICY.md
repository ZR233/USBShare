# USBShare 隐私政策

最后更新日期：2026-03-16

本隐私政策适用于 USBShare（以下简称“本应用”）。USBShare 是一款 Windows 桌面应用，用于通过 SSH 隧道将本地 USB 设备共享到你指定的 Linux 远端并执行 USB over IP attach。

本隐私政策说明：

- 本应用会处理哪些信息
- 这些信息如何被使用、存储和保护
- 信息可能会向哪些对象披露
- 你可以如何管理和删除相关信息

## 1. 我们处理的信息

根据你对本应用的使用方式，本应用可能处理以下信息：

### 1.1 你主动提供的信息

- 远程连接配置，例如远程名称、主机地址、端口、用户名、认证方式、私钥路径、隧道端口
- 你选择保存的 SSH 密码、私钥口令和 sudo 密码
- 你在应用中启用的 USB 设备或 USB Hub 分享规则

### 1.2 本应用在运行过程中读取的设备和系统信息

- 本地 USB 设备、USB Hub、设备实例 ID、Bus ID、设备描述和分享状态
- `usbipd` 相关状态信息
- 应用运行所需的本地配置数据，例如轮询间隔、默认远程目标、自动启动分享设置

### 1.3 发送到你指定远程服务器的信息

当你启用分享功能时，本应用会将与该功能直接相关的信息发送到你自己配置的远程 Linux 主机，包括：

- SSH 连接所需的认证信息
- 为执行 `usbip attach`、`usbip detach`、`sudo`、端口转发和环境探测所需的命令与参数
- 与 USB 分享相关的设备标识信息，例如 Bus ID

## 2. 我们如何使用这些信息

本应用处理上述信息，仅用于实现和维护你主动请求的功能，包括：

- 保存和管理远程连接配置
- 建立 SSH 连接和反向端口转发
- 执行本地 `usbipd bind/unbind`
- 在远程 Linux 主机上执行 `usbip attach/detach`
- 识别和展示本地 USB 拓扑与可分享状态
- 保存你的应用偏好设置，并在需要时自动恢复分享会话
- 排查连接、权限或设备附加失败等功能性问题

本应用当前版本不以内置广告、用户画像、行为分析或面向开发者的遥测上报为目的处理你的个人信息。

## 3. 我们如何存储和保护这些信息

### 3.1 本地存储

本应用主要在你的 Windows 设备本地存储数据。根据当前版本实现：

- 普通配置默认存储在 `%LOCALAPPDATA%\\USBShare\\config.json`
- 保存的敏感凭据默认存储在 `%LOCALAPPDATA%\\USBShare\\secrets\\`

### 3.2 安全措施

- 你保存的 SSH 密码、私钥口令和 sudo 密码会使用 Windows Data Protection API（DPAPI）按当前 Windows 用户进行加密存储
- 本应用通过 SSH 与你指定的远程服务器通信
- 本应用不会默认将你的配置或凭据上传给开发者运营的云服务

请注意，任何软件或网络传输都无法保证绝对安全。如果你的远程主机、SSH 环境、私钥文件、操作系统账号或网络环境本身存在安全风险，相关风险可能超出本应用的控制范围。

## 4. 我们何时共享或披露信息

除以下情况外，本应用不会向开发者、广告商或不相关第三方出售或共享你的个人信息：

- 向你明确配置的远程 Linux 主机传输为实现 USB 分享所必需的信息
- 调用 Windows、`usbipd`、SSH 组件及相关系统能力以完成你请求的功能
- 在适用法律、法规、法院命令或监管要求要求时进行必要披露
- 在保护应用、用户、开发者或公众的合法权益与安全所必需时进行合理披露

第三方软件、服务或系统组件（例如 Windows、SSH、`usbipd-win`、Linux、`usbip`、`sudo`、你自建或托管的远程服务器）可能按照其各自的隐私政策和安全规则处理数据，本隐私政策不覆盖这些第三方的独立行为。

## 5. 数据保留

本应用保留数据的时间通常取决于你自己的使用和管理方式：

- 远程配置和应用设置会保留在本地，直到你修改或删除它们
- 你保存的凭据会保留在本地，直到你在应用中删除对应远程配置，或手动删除本地数据文件
- 当你在应用中删除某个远程配置时，与该远程配置关联的已保存 SSH/sudo 凭据会一并删除

如果你希望彻底删除本应用本地保存的数据，可以删除应用内相关配置，或手动清理 `%LOCALAPPDATA%\\USBShare\\` 目录中的文件。

## 6. 你的控制和选择

你可以通过以下方式控制你的信息：

- 决定是否在本应用中保存 SSH 密码、私钥口令或 sudo 密码
- 编辑或删除远程连接配置
- 修改或关闭自动启动分享、轮询等本地设置
- 停止分享并断开相关会话
- 删除本地配置文件和本地加密凭据文件
- 在你自己的远程 Linux 主机上删除相关日志、授权信息或 USB 附加记录

由于本应用当前主要在本地设备和你自主管理的远程主机上处理数据，很多访问、更正、删除和限制处理请求通常可以由你直接在自己的设备或服务器上完成。

## 7. 儿童隐私
