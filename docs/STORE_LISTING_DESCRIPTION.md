# Store Listing Description

USBShare lets you share locally attached USB devices to a remote Linux machine by using `usbipd-win`, SSH tunneling, and remote `usbip attach`.

Important:

- The app requires administrator permission on launch.
- This is required because `usbipd bind/unbind` needs elevated privileges on Windows.
- Please launch USBShare with administrator permission when installed from Microsoft Store / MSIX.
